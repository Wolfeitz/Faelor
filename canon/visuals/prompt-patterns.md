# Visual Prompt Patterns

These patterns are for direct image generation or external image tools. Replace bracketed fields and preserve canon facts.

## Character

```text
Create [image type] of [character name], [canon identity facts]. Appearance: [fixed visual facts]. Clothing/equipment: [canon outfit/items]. Expression/pose: [pose]. Setting: [location/realm]. Style: follow the project style bible. Apply scopes: global [global style], realm [realm style], race/species/culture [scope style], faction/guild/institution [scope style], medium [medium style]. Lighting/camera: [lighting/camera]. Consistency constraints: [do-not-change facts]. Negative constraints: [avoid].
```

## Map

```text
Create a [map type] for [realm/location]. Required topology: [nodes, routes, boundaries]. Scale/labels: [scale and labels]. Terrain/architecture: [features]. Style: [map style from style bible]. Must preserve: [canon spatial facts]. Negative constraints: [avoid].
```

## Faction Emblem

```text
Create an emblem for [faction]. Symbolic elements: [canon symbols]. Values/themes: [values]. Materials/colors: [palette]. Style: [heraldic/logo/patch/etc]. Must work at small size. Negative constraints: [avoid].
```

## System / UI Message

```text
Create a [message/UI element] for [system, guild, faction, divine source, or realm]. Shared project style: [global style]. Scope-specific style: [race/faction/guild/system layer rules]. Required text or placeholder blocks: [text/slots]. Border/background/icon rules: [rules]. Mood and authority: [tone]. Must remain consistent with: [existing UI/message references]. Negative constraints: [avoid].
```

## Item Or Artifact

```text
Create [image type] of [item name]. Function: [canon function]. Materials: [materials]. Shape/details: [visual facts]. Scale reference: [scale]. Style: [style bible]. Negative constraints: [avoid].
```

## Location

```text
Create an environment image of [location]. Realm/law context: [realm facts]. Architecture/geography: [visual facts]. Mood: [mood]. Inhabitants/activity: [optional]. Style/camera/lighting: [style bible details]. Negative constraints: [avoid].
```

## Scene

```text
Create a scene showing [event/beat]. Characters/objects present: [canon participants]. Location: [location]. Action: [moment]. Emotional tone: [tone]. Style/camera/lighting: [style bible details]. Must preserve: [continuity facts]. Negative constraints: [avoid].
```

## Graphic Novel Page Or Panel Sequence

```text
Create a [page/panel sequence] for [scene or event]. Format: [page size, panel count, gutter style]. Characters: [recurring character continuity facts]. Setting: [location continuity facts]. Action beats by panel: [panel-by-panel beats]. Speech/caption/UI overlays: [balloon, caption, system message rules]. Style scopes: global [global style], realm [realm style], faction/race/guild [scoped style]. Maintain continuity with: [prior page/scene references]. Negative constraints: [avoid].
```
