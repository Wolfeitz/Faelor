# Skills

Optional reusable workflows live here.

Examples:

- provocateur review
- seed extraction
- thing extraction
- export closeout
- integration reconciliation
- source import
- claim extraction
- visual prompt generation
- map validation
- feature packet review

Core workflows:

- `provocateur-review.md`: focused challenge pass before canon promotion, source adaptation, derivation, feature implementation, or major structural decisions.
- `seed-extraction.md`: turns early seeds, source packages, or briefs into candidate stubs, implied infrastructure, open points, and lens recommendations without hardening canon too early.
- `thing-extraction.md`: extracts breadth-first candidate stubs, status, build depth, implied infrastructure, provenance, and promotion gates from chats, sources, drafts, and feature packets.
- `export-closeout.md`: closes major sessions or features with changed artifacts, validation, open points, export needs, and next actions.
- `integration-reconciliation.md`: reconciles overlapping lenses, systems, imports, rules, visuals, and canon areas without assuming fixed modules.
