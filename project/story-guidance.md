# Story Guidance

Status: candidate project guidance
Provenance: extracted from creator discussion.

## Story Role

Fealor is worldbuilding-first, but story development is allowed to drive the world model. The MC, sexual/social tensions, incarnation costs, future Earth friends, and relationship structures are not distractions when they reveal rules, institutions, social pressure, or consequences.

## Mature Tone

The project is not intended to be prudish. Adult sexuality, harem dynamics, sexual tension, jealousy, political marriage, bond mechanics, taboo pressure, and messy relationship consequences can exist in the story if they serve character, society, power, or plot.

The workspace should preserve these intentions as project context even when a specific passage cannot be drafted directly by the assistant.

Practical guardrails and response posture are tracked in `project/guardrails.md`.

## Sensitive Placeholder Strategy

When a scene crosses assistant drafting boundaries:

1. Preserve the scene's purpose.
2. Preserve the before/after continuity.
3. Add a neutral placeholder.
4. Record the intended emotional, social, or plot consequence.
5. Continue drafting the surrounding material.

Use this shape:

```markdown
[SCENE PLACEHOLDER: sensitive passage]

Purpose:
- What this scene must establish.

Exit state:
- What must be true afterward.

Continuity notes:
- What future scenes need to remember.
```

## Newborn Incarnation Tone

Newborn incarnation should not be sanitized. It carries dependency, lack of dignity, privacy loss, caregiver intimacy, cultural mismatch, helplessness, identity dissonance, and later puberty/adult-boundary complications.

In this workspace, infant/child/adolescent material is treated as dependency, identity dissonance, vulnerability, social complication, and long-term emotional consequence. Adult sexuality can be developed later for adult characters.

## Proactive Guidance

The agent should not merely ask "what next?" after every brainstorm. It should propose the next useful worldbuilding operation, usually one of:

- extract stubs;
- run a Provocateur pass;
- promote selected anchors;
- deepen implied infrastructure;
- build a first scene/world interface;
- create character translation profiles;
- create a candidate canon file.
