# Requirements

## Functional Requirements

- The base scaffold must support output-format selection and export closeout.
- The base scaffold must support advisory routing when a thing has multiple possible facets.
- The base scaffold must support optional system profiles for visibility, progression, interface, integration, and output needs.
- The base scaffold must support reconciliation between overlapping lenses, systems, sources, visuals, rules, and feature packets.
- Older fixed module concepts must be generalized into optional lenses and workflows.

## Constraints

- Do not hardcode old Kadzu module categories.
- Do not assume stats, levels, UI, divine influence, AI interfaces, cultivation, or LitRPG.
- Preserve zero-assumption behavior.
