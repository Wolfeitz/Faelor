# Brief

## Purpose

Extract reusable project-agnostic value from older Kadzu system-engine documents without importing fixed module assumptions.

## Scope

- Add export and closeout guidance.
- Add facet routing.
- Add optional system profile modeling.
- Add integration reconciliation.
- Add session/export closeout workflow.

## Non-Goals

- No hard Kadzu module taxonomy.
- No assumption that projects have stats, levels, UI, LitRPG systems, or game mechanics.
- No fixed attribution footer.
