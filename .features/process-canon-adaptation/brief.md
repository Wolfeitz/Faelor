# Brief

## Purpose

Make Lughbrugh adapt across many project types by treating creator guidance, constraints, tone, placeholder strategy, and repeated workflow corrections as durable process canon.

## Scope

- Add project guardrails.
- Add creative guidance.
- Add project lessons.
- Add formal thing extraction.
- Standardize thing status and build depth.
- Wire Provocateur and agent instructions to recommend guidance updates.

## Non-Goals

- No project-specific lore.
- No fixed genre, ruleset, medium, or moral tone.
- No automatic rewriting of project guidance without creator confirmation.
