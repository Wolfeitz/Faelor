# Decisions

- Creator preferences, guardrails, creative posture, and workflow lessons are process canon.
- Process canon is durable operating material, not world canon.
- Thing extraction should be breadth-first before depth-first.
- Implied infrastructure should be captured as candidate stubs.
- Placeholders preserve continuity when exact content is deferred, private, blocked, source-limited, or better handled later.
- The Provocateur can recommend project guidance updates but should not silently rewrite them.
