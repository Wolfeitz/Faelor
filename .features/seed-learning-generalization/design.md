# Design

## Seed Extraction

Early seeds often contain more structure than they appear to. The workflow extracts lightweight candidates with known facts, inferred facts, open points, implied infrastructure, possible lenses, build depth, and promotion gates.

## Interface Abstraction

Many projects expose truth through a display or mediation layer. The lens separates interface from mechanism so agents do not mistake a convenient UI, ritual, title, menu, or social convention for objective reality.

## Cross-Context Translation

When material moves between contexts, the scaffold tracks what carries, changes, locks, fails, or requires local grounding. This supports source adaptation, rules conversion, portal-style stories, imported corpora, visual style inheritance, and setting migration without assuming any one genre.

## Reuse Boundary

The reusable layer may capture tools, checks, workflows, and modeling patterns. Project-specific lore remains in the project that created it.
