# Brief

## Purpose

Capture reusable lessons from seeded project development without importing project-specific lore.

## Scope

- Add project-agnostic seed extraction guidance.
- Add interface abstraction guidance.
- Add cross-context translation guidance.
- Record reusable lessons in project guidance.
- Wire the new workflows into agent instructions.

## Non-Goals

- No project-specific canon.
- No setting-specific mechanics.
- No hard ontology.
