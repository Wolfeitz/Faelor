# Brief

## Purpose

Make proactive forward prompting a default Lughbrugh behavior so creators are not forced to pull momentum from the agent every turn.

## Scope

- Update creator preferences.
- Update operating mode.
- Update personality and agent instructions.
- Record the behavior as a project lesson.

## Non-Goals

- Do not allow agents to silently author canon.
- Do not require long next-step menus after tiny factual answers.
- Do not override explicit creator requests for narrow answers.
