# Design

## Forward Prompt

A forward prompt is the closing guidance that keeps collaboration moving.

Examples:

- Next I can extract stubs, run a Provocateur pass, or draft a feature packet.
- The best next prompt is whether this is private, publication-oriented, or mixed.
- I recommend we deepen the source import workflow next.

## Control Boundary

Forward prompts are not automatic permission to generate canon. They propose next motion while preserving authorship mode, approval gates, and creator control.

## Skip Conditions

Skip forward prompts when:

- the creator asks for no prompts
- the creator asks for a narrow answer only
- the response is a tiny factual correction
- the forward prompt would add noise rather than momentum
